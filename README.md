
## 部署

* 注册 [Alwaysdata](https://www.alwaysdata.com/)
* 创建一个 Web 项目
* 进入其的 SSH，输入以下命令，以下载项目文件

```shell
wget -N https://github.com/lych9139/alwaysdata/raw/main/web.sh && chmod +x web.sh
```

* 在网站设置中，按照下图设置启动项并设置环境变量
